# Protocol Repository Status

Thank you for your interest in the SemaBuzz Protocol engine!

**Notice: This repository is currently maintained as a read-only open-source release.**

At this time, we are **not** accepting Issues or Pull Requests. The repository is provided so that developers can read, audit, and understand the protocol.

## Forking and Modifications

SemaBuzz Protocol is open-source under the **GNU AGPL v3.0** license. 

While we are not accepting upstream contributions, you are completely free to fork this repository, modify the code, and build your own applications based on the protocol, provided you comply with the terms of the AGPL v3.0 license.

## Building Locally

```bash
git clone https://github.com/skynrlabs/SemaBuzz-Protocol.git
cd SemaBuzz-Protocol
dotnet build
dotnet test
```
